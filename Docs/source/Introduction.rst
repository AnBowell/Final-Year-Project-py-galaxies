Introduction to PyGalaxies
==========================

.. toctree::
   :maxdepth: 2
   :caption: Contents:


The aim of this project is to write a simple but efficient front for L-Galaxies.

This package like structure allows jupyter notebooks to call classes/functions and run simulations.

Source code is located `here <https://github.com/AnBowell/Final-Year-Project-py-galaxies>`_
