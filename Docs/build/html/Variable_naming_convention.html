Variable Conventions¶

PyGalaxies Variable Naming Convention¶

Description and type of variable

Naming Convention

Standard variable. First word > 2 chars.

<variable_name>

Constant (generally for local use)

<VARIABLE_NAME>

Boolean

b_<name>

Loop counter

i_<name>

Total count of <name> objects.

n_<name>

Class name

<Name>

Python class definition of <Name>

C_<name>

Python dictionary

D_<name>

Python list

L_<name>

A Python tuple

T_<name>

A Python function

F_<name>
